# 🇩🇪 Bundeshaushalt

Tool to scrape the German Budget.


## ⬇️Install
- Clone the repository and `cd` to it
- `yarn install` or `npm install`

## 🏃Running
```node scrape.js```
This will create a `functions.json` file that contains all the `Funktionen` for the haushalt
